# Autonoma public feedback discussions

In this repository, you can find the [official Autonoma public feedback discussions here](https://github.com/autonomauk/feedback/discussions).

These discussions are where you can share suggestions for how the service should be improved and discuss those improvements with the community.

## Making suggestions

We encourage you to [open a discussion](https://github.com/autonomauk/feedback/discussions) if you have suggestions for how we can improve our products. You don't need to have a solution to the problem you are facing to kick off a discussion. We are hoping to foster productive and collaborative conversations, so please check out [how to give good feedback from GH themselves](https://github.com/github/feedback/discussions/1) if you want some guidance on how to kick off a successful discussion.

Prior to making a new discussion, please take a look at previous discussions to see if someone else has already brought to our attention your suggestions. If you find a similar discussion, reply with additional details or upvote the discussion to signal your support rather than creating a new discussion.

### From a suggestion to a shipped feature

Once you kick off a discussion the Autonoma product team will evaluate the feedback but will not be able to respond to every submission. - If the team determines that we will not be working to solve the problem you have identified, we may comment on the discussion describing our reasoning so our decisions can remain transparent.
